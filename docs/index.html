Overview

Mini 4WD Converter is a comprehensive web application designed to convert various file formats (STL, CAD, SVG, JPG, PNG) specifically for Tamiya Mini 4WD parts. The application features multi-format upload capabilities, CAD/STL output, Mini 4WD parts validation with error handling, an intuitive drag-and-drop interface, real-time feedback, and responsive design.

Getting Started

Quick Start

Get the Mini 4WD Converter running in minutes:

# Clone the repository git clone https://github.com/lizewkn/mini4wd-converter.git cd mini4wd-converter # Start the backend cd backend pip install -r requirements.txt python app.py # In a new terminal, start the frontend cd frontend npm install npm start # Or use the standalone interface open standalone/index.html

System Requirements

Backend Requirements

  • Python 3.8+
  • Flask framework
  • NumPy, Pillow libraries
  • Trimesh for 3D processing

Frontend Requirements

  • Node.js 16+
  • React 18
  • TypeScript
  • Tailwind CSS

API Reference

Base URL

http://localhost:5000/api

Endpoints

POST /upload

Upload a file for processing

curl -X POST \ http://localhost:5000/api/upload \ -F "file=@example.stl"

POST /convert

Convert uploaded file to specified format

curl -X POST \ http://localhost:5000/api/convert \ -H "Content-Type: application/json" \ -d '{ "file_id": "uploaded_file_id", "output_format": "stl", "validate_mini4wd": true }'

GET /download/{file_id}

Download converted file

curl -X GET \ http://localhost:5000/api/download/converted_file_id \ -o output.stl

Usage Examples

Converting an STL File

# 1. Upload the file curl -X POST -F "file=@wheel.stl" http://localhost:5000/api/upload # 2. Convert to OBJ format curl -X POST \ -H "Content-Type: application/json" \ -d '{"file_id":"uploaded_id","output_format":"obj"}' \ http://localhost:5000/api/convert # 3. Download the result curl -X GET http://localhost:5000/api/download/converted_id -o wheel.obj

Validating Mini 4WD Compatibility

# Upload and validate a chassis curl -X POST \ -H "Content-Type: application/json" \ -d '{"file_id":"chassis_id"}' \ http://localhost:5000/api/validate

Deployment

Docker Deployment

# Build and run with Docker Compose docker-compose up --build # Or build individual containers docker build -t mini4wd-backend ./backend docker build -t mini4wd-frontend ./frontend
